# m3u8视频下载工具
一个m3u8视频下载工具，支持并发下载
## 基本使用
![Peek 2023-02-28 18-20](https://user-images.githubusercontent.com/21306882/221826035-8c1bd85b-7058-438a-8801-6d7ca2091388.gif)
## 参数说明
```
  -f string
    	保存的文件地址 (default "movie.mp4")
  -m int
    	并发数
  -m3u8 string
    	m3u8下载链接
  -mem
    	是否内存加速，会占用大量内存
```
